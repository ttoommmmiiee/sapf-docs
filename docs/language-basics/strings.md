# Strings


Strings are enclosed in double quotes. 

```
sapf> "This is a string"
This is a string
```

\\n is newline and \\t is tab.

```
sapf> "\\tThis string begins with a tab and ends with a newline.\\n"
        This string begins with a tab and ends with a newline.

sapf> 
```
